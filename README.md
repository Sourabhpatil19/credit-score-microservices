# Credit Score Microservices System

## 🚀 Overview
This project is a microservices-based Credit Score Analysis Tool designed to evaluate customer credit scores in real time.

## 🏗️ Architecture
Client → API Gateway → Microservices → Database

## 📦 Services
- User Service
- Data Service
- Score Service
- Report Service
- API Gateway

## 🛠️ Tech Stack
- Java 21
- Spring Boot
- MySQL
- Kafka (Planned)
- Redis (Planned)
